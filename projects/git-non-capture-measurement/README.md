# Git Non-Capture Measurement

Status: Lane 1 first-experiment scaffold.

## Purpose

This directory prepares the charter's first experiment: measuring whether this
repository's own git substrate makes capture cost non-decreasing with
accumulated legitimacy and participation, or whether observed resistance is
actually traceable to founder decisions.

This is the Lane 1 force-or-falsify path. It is not a contribution-workflow
pilot, governance upgrade, public result, or proof that the North Star holds.

## Protected Boundary

Artifacts here may define measurement fields, admissibility checks, founder
discretion labels, source boundaries, and output shapes.

They may not:

- issue a legitimacy verdict;
- declare K1 fired or cleared;
- change the charter, Lane manifest, governance, or claim status;
- adopt capture mitigations, reviewer powers, rights, rewards, or workflow
  policy; or
- publish or represent a result outside normal GitHub versioning.

## Measurement Discipline

The charter requires this repository to measure the host first. Every later
artifact in this directory should keep three columns separate:

- what git itself makes possible or costly;
- what accumulated repository legitimacy or participation changes; and
- what Joe or founder-led governance can decide directly.

A finding that capture resistance comes from founder discretion rather than git
properties is an admissible adverse result, not a problem to repair during the
run.
