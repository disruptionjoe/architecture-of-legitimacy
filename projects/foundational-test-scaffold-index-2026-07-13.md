## Purpose

This index makes the older foundational test scaffolds easier to find from one
place after the repo index coverage audit showed uneven repo-level literal-path
coverage.

change, public-posture change, review launch, source-selection decision, source
fact fill, contribution record, reward/right meaning, or instruction to promote
any artifact.

scaffolds behind the current review-prep stack. For current lane order, use

## Reading Notes

- Read these as foundation and provenance for later first-pilot and
  second-ring review-prep artifacts, not as the current state of all review
  questions.
- If a later synthesis conflicts with an older scaffold, treat the conflict as
  a review question rather than silently promoting either surface.
- Do not infer readiness from index inclusion. Inclusion only means the artifact
  is useful enough to be discoverable.
