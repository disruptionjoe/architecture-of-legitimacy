# Project Surface Map

Status: navigation surface map.

## Purpose

This directory contains bounded research scaffolds for the project's agenda and
test matrix. The files here make contribution, legitimacy, reward, governance,
capture, AI, and transfer questions inspectable before any live review or policy
movement.

This map is navigation only. It does not change claim status, test status,
canon verdicts, contribution standards, contribution records, governance,
rights, rewards, AI policy, transfer posture, public posture, or live-review
state.
