# First-Ring Synthesis Surface Map

Status: navigation surface map.

## Purpose

This directory contains non-adopted first-ring synthesis scaffolds. The files
here make the RQ1-RQ5 review-prep burden, later first-pilot deltas, and
protected non-movement boundaries easier to read before any live contribution
workflow or policy movement.

This map is navigation only. It does not launch review, select a target, fill a
packet, classify a real contribution, adopt contribution classes or review
paths, assign reviewer authority, change claim status, change test status,
adopt governance, create rights or rewards, change AI policy, alter transfer
posture, move public posture, or create live contribution records.

- selecting a real first-pilot target, contributor, neighboring system, source
  lane, live record surface, reviewer owner, or public audience;
- filling a source-trace packet or changing a synthetic packet into a live
  contribution object;
- opening an issue, pull request, review thread, public invitation, or
  contribution-log entry;

  review policy, governance files, guardrails, claims, or test-status surfaces;

  duties, cadence, appeal rights, dispute rules, reviewer authority, response
  time promises, reward logic, rights, governance rules, AI policy, transfer

  conclusions, test outcomes, claim-status movement, sanctions, monitoring, or
  contribution-record meaning;
