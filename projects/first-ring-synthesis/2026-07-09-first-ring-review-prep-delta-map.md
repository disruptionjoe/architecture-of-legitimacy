## Updated Review-Prep Read Order

A later governed reviewer should read the first-ring material in this order:

1. Read the original first-ring crosswalk for the broad trace burden.
2. Read the RQ5 review-prep bundle map for the source-trace artifact order.
3. Read the source-trace revision state map before interpreting missing fields.
4. Read the review-question register for unresolved governed questions.
5. Read the RQ2 addendum before allowing value or rationale discussion to
   proceed past classification uncertainty.
6. Read the value-disagreement and reviewer-capacity ledgers before any score,
   owner, cadence, or live window language appears.
7. Read the diagnostic-residue map before using legitimacy-failure language.

The order is conservative. It keeps field completeness, class disagreement,
value disagreement, reviewer capacity, and diagnostic residue visible without
letting any one artifact clear the others.
